# 모두의 스터디
---
#### ◼ 프로젝트 개요
스터디를 모집하고 함께 창작물을 산출해보는 서비스가 있다면 어떨까? 

👉🏻 본 프로젝트는 스터디 그룹을 모집하고, 스터디 활동 결과물인 나무위키 형식의 문서를 작성하여 참여자들의 포트폴리오로 활용할 수 있는 웹 사이트 제작을 목표로 합니다.

#### ◼ 기대 효과 🤼
- 온/오프라인 스터디 그룹 형성 및 활동 지원
- 협업을 통한 지식 공유 및 문서화 작업 수행
- 참여자들의 포트폴리오 제작 지원
